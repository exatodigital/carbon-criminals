# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Propósito

Repositório de zueira interna da Exato Digital para ranking mensal de uso do Claude Code, com temática de "destruição ambiental".

## Estrutura

- `RANKING_DESTRUICAO_AMBIENTAL.md` - O ranking principal com GIFs e emojis
- `claude_code_team_*.csv` - Dados de uso exportados do Claude Code

## Atualizações

Para atualizar o ranking, basta editar o arquivo `.md` com os novos dados do CSV mensal.
